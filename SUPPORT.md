# Trace — Support

Thanks for using **Trace**.

Trace is a **local-first macOS app** that helps you search what you’ve seen on your screen.  
Your data stays on your Mac.

---

## Quick Help

### The app can’t capture my screen
Trace needs **Screen Recording** permission.

1. Open **System Settings → Privacy & Security → Screen Recording**
2. Enable Trace
3. Quit and reopen Trace

### Search feels empty / no results
- Wait a bit so Trace can capture and index new items
- Make sure Screen Recording permission is enabled
- If you recently enabled permissions, restart the app

### OCR text looks wrong / incomplete
OCR depends on what’s visible on screen, fonts, and contrast.  
Try:
- Increasing window size / zoom
- Waiting for the next capture cycle
- Checking that your screen content is not blurred (e.g. low-res remote sessions)

### Where is my data stored?
Trace stores data locally inside your app container (Application Support).  
There is no cloud sync.

## Contact

If you need help or want to report a bug:

- Open an issue here: **GitHub Issues**
- Include:
  - macOS version
  - Trace version
  - What you expected vs what happened
  - Screenshots/logs (remove sensitive info)

Trace is evolving fast. If you have an idea, open an issue and describe:
- the problem you’re solving
- your ideal workflow
- example screenshots or mockups (optional)

## Disclaimer

Trace can capture information visible on your screen.  
Use it responsibly, and avoid capturing sensitive screens if you don’t want them stored locally.
