# Trace

**Your Mac’s local memory.**  
Search what you’ve seen — fast, private, and on-device.

---

## ✨ Overview

Trace captures what appears on your screen and turns it into a searchable, structured memory system.

Instead of losing context across screenshots and apps, Trace helps you instantly find past information — without relying on cloud services.

---

## 🚀 Key Features

- 📸 Automatic screen capture (with permission)
- 🔍 OCR text extraction
- 🧠 Semantic + keyword search
- 🗂 Session-based organization
- 🔒 Fully local, privacy-first

---

## 🔐 Privacy-first by design

- No cloud sync  
- No tracking  
- No accounts  
- All data stays on your Mac  

---

## 🖥 Requirements

- macOS 14+
- Screen Recording permission enabled
